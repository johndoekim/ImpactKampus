# 일상의 권태기를 탈출 시킬 사소한 미션 추천 서비스 포미 

제한 시간이 있는 미션을 수행함으로써, 성취감을 얻고 이를 기록하며 자신을 돌아보는 서비스

---

## 주요 기능

짧은 시간 내에 수행 할 수 있는 기본적인 시스템 미션을 부여
유저 간 미션을 부여 할 수 있는 기능을 통해 재미를 기대
